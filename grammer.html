Quick Reference

Open vs Closed

Open class grows (new slang/tech words). Closed class is limited (e.g., prepositions like in, on, at).

Word families

Create (verb) → creative (adj) → creatively (adv) → creation (noun).

Urdu Tip

Verb aksar kaam ya harkat batata hai (جیسے: run, think). Adjective sifat batata hai (خوبصورتی، بڑا).
